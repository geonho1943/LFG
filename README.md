## Environment
LFG 서비스는 이것으로 실행 됩니다.
* Language: Java17
* Framework: Spring Boot 2.7.3
* DB: MariaDB 5.5.68
* Template engine: Thymeleaf
* JavaScript Library: jQuery 3.6.0

LFG 서비스는 이것으로 만들어졌습니다.
* IntelliJ IDEA Community Edition 
* Visual Studio Code
* Spring Initializr

LFG 서비스는 이곳에서 실행됩니다
* AWS Ec2
* Oracle Cloud
